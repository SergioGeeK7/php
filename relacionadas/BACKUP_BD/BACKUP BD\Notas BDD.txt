Como relacionar tablas >> conjunto de base de datos relacionadas entre si


Modelo entidad relacion

tres entidades>> autor libro genero

------------
autor >> libro >> genero >>>>>>>>>>>>  relaciones
------------
dos tablas se relacionan por campos en comun 

1. con el mismo significado (puede no se comunes aun asi tengan el mismo nombre en las tablas)
2. con el mismo tipo de dato

--

como deben ser esos campos en comun?

un campos debe ser la clave primaria de una tabla y el otro campo debe ser un campo no clave primaria de la otra tabla

--

campo que signifique lo mismo que la primary key de la otra tabla...
(no se relaciono con codigo codigo porque ambas son primary key)

SE DEFINIE CON QUIEN TIENE A QUIEN.... EN EL ORDEN QUE SE ASIGNA LA RELACION

MULTIVALORADO NO ES (SE DESCARTA);
UNIVALORADO ES (SE CREA EL CAMPO);


CAMPO ATRAVES DEL CUAL LA TABLA LIBRO SE RELACIONA CON LA TABLA LIBRO... SE CONOCE COMO CLAVE FORENEA... (PUEDE SER NULL)

CLAVE PRIMARYA DE UNA TABLA Y LA CLAVE FORENEA DE OTRA


----------
	1		      M
AUTOR >>>>>>>>>>>>>>>>>>>>>>>>> LIBRO    >>>>>>>>>> MUCHOS A MUCHOS (MUCHOS AUTORES HACEN LIBROS MUCHOS LIBROS TIENEN VARIIOS AUTORES)
	M		      M

TABLA INTERMEDIA QUE CONTIENE DOS CAMPOS; >>>  LA PRIMARY KEY DE AUTOR Y LA PRIMARY KEY DE LIBRO

ALTER TABLE LIBRO

-- COMO RELACIONAR LA TABLA LIBRO Y GENERO ? 
--COMO CREARLE A LA TABLA LIBRO LA CLAVE FORENEA

ALTER TABLE LIBRO -- ADD CLAVE FORANEA
ADD FOREIGN KEY (CODGENERO)	
REFERENCES GENERO(CODIGO) -- CODIGO ES PRIMARY KEY DE LA TABLA 'GENERO'
--WARNING QUE NO HAYA LIBROS CON GENEROS QUE NO EXISTAN EN CODGENERO

--PARA LA RELACION ENTRE LIBRO Y AUTOR , COMO ES
--UNA RELACION MUCHOS A MUCHOS, SE GENERA UNA NUEVA TABLA
-- CON LAS CLAVES PRIMARIAS DE LIBRO Y AUTOR.

CREATE TABLE LIBAUTOR
(CEDAUTOR BIGINT,
CODLIBRO BIGINT,
PRIMARY KEY(CEDAUTOR,CODLIBRO))  -- CUANDO SE QUIERE ESTABLECER DOS PRIMARY KEY SE HACE AL FINAL
-- TIENE UNA PRIMARY KEY COMPUESTAS DE DOS CAMPOS
-- CUANTAS CLAVES FORANEAS TIENE LIBAUTOR 2 UNA DE UNA Y OTRA DE OTRA
